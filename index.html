Návod na manuálnu inštaláciu (Developer Mode)

  1. Stiahnite alebo klonujte tento projekt do počítača.
  2. Otvorte Chrome a prejdite na chrome://extensions.
  3. Vpravo hore zapnite Developer mode.
  4. Kliknite na Load unpacked.
  5. Vyberte koreňový priečinok rozšírenia: TabNest.
  6. Ak bude rozšírenie neskôr vyžadovať API key, vložte ho v nastaveniach rozšírenia.
